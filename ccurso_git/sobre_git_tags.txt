Comando para criar tag 'git tag -a "Nome da tag que quer criar" -m "descrição da tag"'
Comando para deletar tag 'git tag -d "nome da tag"
Comando para ver a tag criada 'git tag'
Comando para ver o que tem em cada tag 'git show "nome da tag"'
Comando para alternar entre tags 'git checkout (igual às branches) "nome da tag"'
Comando para enviar tag específica para Github 'git push origin "nome da tag"'
Comando para enviar todas as tag para Github 'git push origin --tags'
Comando para clonar o arquivo (vai pro github e copia o código do arquivo) 'git clone "link do arquivo" .'
(lembrar do espaço . do comando acima!)
Comando para dar push em arquivo clonado 'git push --set-upstream origin "nome da branch"'
Comando para buscar branches que não estão no programa atual 'git fetch -a'
Comando para dar mais performance para nosso repositótio 'git gc'
Comando para ver a linha do tempo do projeto 'git shortlog'
Comando para comparar arquivos modificados com os que estão no GitHub 'git diff'
Comando diff para arquivo específico 'git diff HEAD: "nome do arquivo"'
Comando para mostrar em detalhes as mudanças feitas no repositório (nos últimos 30 dias)'git reflog'
