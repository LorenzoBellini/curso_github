"git add ." já adiciona todos os arquivos da pasta!
"git commit 'arquivo' -m "TEXTO" " - comita o arquivo com um comentario
"git commit  -m "TEXTO" " - comita todos os arquivos com um comentario
"git status" mostra se tem algum arquivo que sofreu alteração e precisa ser 'comitados'
'git add' antes de 'git commit'
'git status' antes e/ ou depois
'git push' pra enviar o arquivo pro GitHub
'git pull' para puxar arquivos que foram feitos no git mas não estão na máquina
'git log' para receber todas as mudanças e atualizações do arquivo/ programa
